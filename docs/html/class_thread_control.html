bool ThreadControl::startSystem ( void  )

Master function to begin spinning up several thread process from a single start call orginating from the main function. Controls setting up varaibles based on config values. Kick starts the threads and sets them (via main) into the process loop for Qt.

Is using Qt based POSIX threading to avail cross platform porting if needed.
